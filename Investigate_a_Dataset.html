#  instead of write code every time i created function and

def bar_plot_fun(heights_low, heights_high, tit, xlab, ylab):
    locations = [1, 2]
    heights = [heights_low, heights_high]
    labels = ['Low', 'High']
    plt.bar(locations, heights, tick_label = labels)
    plt.title(tit)
    plt.xlabel(xlab)
    plt.ylabel(ylab)

Conclusions¶

We removed the columns that we don't need in analysis.

We removed the null values from rows, we found that rows that have null values is not very much so we romoved them.

We removed duplicated rows, there is just one row is duplicated.

After we cleaned data we started on exploratory the data and answaring questions.

Fist: We found that movie that has high revenue has high popularity.

second: We found that the movie that achieves high revenues also achieves a high rating, but not at a high rate.

Third: We found that the movie that has a high budget also has a high revenue.

Fourth We found that long duration for movies, for example, that exceed 90 minute have a high revenue and high movie rating.

Fifth: we get in each year which genres is most popular.

Sixthly: We found that Drama movies has the most popularity.

Limitation of Dataset 1- Some variables of dataset included a great deal of null data that may result in misleading insights when analyzing, such that homepage column so i opted to delete. 2- I removed some columns that i'w not use them in analysis, such that id, imdb_id, cast, director, tagline, keywords and overview. 3- Budget and revenue columns have many zeros value.
